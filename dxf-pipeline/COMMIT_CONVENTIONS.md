# 提交信息约定

推荐格式：

```text
<type>(<scope>): <summary>
```

## type 建议

- `feat`：算法能力或新导出能力
- `fix`：几何/路径/轨迹/契约修复
- `refactor`：结构重整
- `test`：测试与基准相关
- `docs`：文档变更
- `build`：脚本、生成流程变更
- `chore`：杂项维护

## scope 建议

- `geometry`
- `path`
- `trajectory`
- `contracts`
- `proto`
- `adapters`
- `tests`
- `docs`

## 示例

- `feat(trajectory): 增加采样报告字段`
- `fix(proto): 修复兼容性回归样例`
- `refactor(path): 拆分路径排序与分段逻辑`
