## 📋 审查制度

### 审查周期

- **频率**: 每季度（每3个月）一次
- **时间**: 1月、4月、7月、10月的第一个周
- **负责人**: @lead-architect
- **参与者**: 技术委员会 + 各模块负责人

### 审查清单

- [ ] 检查ADR状态是否仍准确（accepted/deprecated/superseded）
- [ ] 检查实施状态（0% / 50% / 100%）
- [ ] 评估决策的有效性（是否达到预期目标）
- [ ] 识别需要更新的ADR
- [ ] 记录新的替代方案或技术变化
- [ ] 更新下次审查日期
