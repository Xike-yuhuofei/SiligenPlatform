# SiligenSuite Skills Index

本目录提供项目内 Codex 技能，统一采用 `ss-` 前缀。

所有技能默认依赖：

1. `tools/scripts/resolve-workflow-context.ps1`（统一 `ticket/branchSafe/timestamp`）
2. `tools/scripts/invoke-guarded-command.ps1`（高风险命令守卫）

## 主流程技能

1. `ss-scope-clarify`：需求澄清与范围锁定
2. `ss-arch-plan`：架构与测试计划
3. `ss-premerge-review`：预合并代码审查
4. `ss-qa-regression`：测试-修复-回归
5. `ss-release-pr`：发布与 PR 提交
6. `ss-doc-sync`：文档同步
7. `ss-weekly-retro`：周度复盘

## 按需技能

- `ss-ui-plan-check`：UI 方案审查（HMI）
- `ss-incident-debug`：故障定位与根因分析
- `ss-safety-guard`：高风险操作护栏
- `ss-workflow`：流程路由与技能选择
