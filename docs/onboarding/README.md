# Onboarding

本目录承接开发者和代理进入仓库时需要的正式入场文档。

适用内容：

- 默认开发工作方式
- 根级构建、测试、验证入口
- 分支、worktree、协作约定
- 新成员或新代理开始工作前应先读的基础规则

当前建议入口：

- [developer-workflow.md](./developer-workflow.md)
- [workspace-onboarding.md](./workspace-onboarding.md)

使用规则：

1. 当 `Codex` 需要快速进入仓库默认工作方式时，优先读取本目录
2. 本目录是正式指引，不应混入单次任务 handoff 或临时排障记录
