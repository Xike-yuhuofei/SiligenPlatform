# Archive

本目录默认属于历史文档区。

它承接已退出主流程、已被替代或仅保留追溯价值的旧方案、旧阶段产物和归档材料。

## 子目录

- `2026-03/`：2026 年 3 月归档的历史版本材料，包括 hmi-client 旧文档
- `process-model/`：从 `process-model/` 精简移出的临时批次计划和多版本评审记录
- 顶层零散文件：其他被取代的历史文档（如 `legacy-cleanup-plan.md`）

## 使用规则

1. 本目录不提供当前正式口径
2. 仅在需要追踪历史演进、比对旧方案或保留审计痕迹时读取
3. 若归档材料中仍有当前有效的结论，应上收至正式文档区，而不是继续依赖归档副本
