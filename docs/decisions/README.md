# Decisions

本目录承接 `SiligenSuite` 工作区级正式决策文档。

适用内容：

- ADR
- 关键治理决策
- 边界冻结结论
- 对当前仓库仍然有效的演进选择记录

使用规则：

1. 当需要判断“为什么当前采用这套规则”时，优先读取本目录
2. 本目录记录的是结论与理由，不承担任务过程追踪职责
3. 若某篇文档只描述阶段性讨论、候选方案或临时工作计划，应放到历史文档区，而不是本目录
