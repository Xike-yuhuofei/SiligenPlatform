# Machine Model History

本目录保存 `machine-model/` 下不再承担当前机型正式真值职责、但仍保留追溯价值的历史资料。

使用规则：

1. 先读取 [docs/machine-model/multicard-io-and-wiring-reference.md](../multicard-io-and-wiring-reference.md) 作为当前机型 authority。
2. 仅当需要追溯历史诊断方法、旧工具说明或迁入副本的历史背景时，才进入本目录。
3. 若历史材料与当前机型事实冲突，以正式区为准。
