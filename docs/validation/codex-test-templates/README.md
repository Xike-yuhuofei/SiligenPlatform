# Codex 测试指令模板

本目录收口可直接复制给 Codex 使用的测试指令模板。

适用场景：
- 无机台验证
- mock / 仿真联调
- 功能回归测试
- 单功能深挖测试

文件列表：
- `01-无机台测试最小模板.md`
- `02-无机台测试推荐模板.md`
- `03-无机台测试批量扫描模板.md`
- `04-回零功能专项模板.md`
- `05-阀控制专项模板.md`
- `06-无机台强执行模板.md`
- `07-功能回归测试模板.md`
- `08-单功能深挖测试模板.md`

使用原则：
1. 优先选择最贴近当前任务范围的模板，不要默认使用最大模板。
2. 如果目标是“让 Codex 直接执行”，必须保留“不要只分析，直接执行”的要求。
3. 如果目标是“先复用现有资产”，必须保留“优先复用仓库内已有 mock/smoke/unit test/ctest”的要求。
4. 如果目标是“尽量不改业务代码”，必须保留“先不要改业务代码，只有覆盖不足时才补最小测试”的要求。
