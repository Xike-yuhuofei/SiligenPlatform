# 无机台测试推荐模板

```text
请对以下功能做“无机台验证”，执行方式参照你刚才对点动功能的做法：

功能：
1. 回零
2. 点胶阀开/关
3. 供胶开/关
4. 急停
5. move/move_to
6. 安全门/急停互锁对上述功能的拦截

执行要求：
1. 不用真实机台。
2. 优先查找仓库中已有的 mock、smoke、unit test、ctest、协议测试入口。
3. 先跑现有测试；现有测试不足时，再补最小测试。
4. 不要只做分析，直接执行测试。
5. 每完成一项，汇报：
   - 用了哪些脚本/命令
   - 关键输出和结论
   - 是否真正覆盖到该功能主链路
6. 最后给我一份汇总表：
   - 功能
   - 测试方式
   - 结果
   - 风险/未覆盖项
```
