# 无机台测试最小模板

```text
请像你刚才测试点动功能一样，继续用“无机台 + mock/仿真”的方式测试以下功能：

1. 回零
2. 点胶阀控制

要求：
1. 不使用真实机台，不连接真实硬件。
2. 优先复用仓库里现有的 mock server、smoke 脚本、单元测试、集成测试，不要先改代码。
3. 如果现有测试不够，补充最小必要的测试，并实际执行。
4. 分别给出：
   - 测试入口命令
   - 实际执行结果
   - 通过/失败结论
   - 当前无机台测试能覆盖什么，不能覆盖什么
5. 如果某项测试因为缺少构建产物、脚本或环境而无法执行，要先定位原因，再尝试补救；补救不了再告诉我。
6. 全程使用中文。
```
