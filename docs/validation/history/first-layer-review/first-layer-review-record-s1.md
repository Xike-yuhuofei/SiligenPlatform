## 一、场景说明

- 业务前提：默认应包括设备连接成立、回零完成、配方已激活、DXF 输入有效
- 验证上下文：应能进入可执行主干任务的有效就绪态
- 关键输入条件：应明确本次使用的 DXF、配方版本、关键参数与启动条件
- 证明责任摘要：证明“设计输入 → 路径规划 → 点胶执行 → 结果输出”整条主链可在脱机环境下闭合
- 成功边界：主干流程完整闭合，结果可输出、可观察、可判定
- 失败边界：只能到达局部环节、无法形成结果输出、或结论仍依赖人工解释
