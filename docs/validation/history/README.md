# Validation History

本目录承接验证体系中的历史材料。

读取规则：

1. 先看 `docs/validation/README.md` 中列出的正式文档
2. 只有在需要追溯某次阶段性执行、incident 修复验证或首层评审过程时，才进入本目录
3. 若本目录中的结论仍需长期生效，应上收至正式区，而不是继续把历史记录当真值

当前分组：

- `first-layer-review/`：首层评审和复审过程工件
- `incidents/`：incident/bug 的单次验证 closeout
- `dxf/`：DXF 阶段性回归和会话总结
- `phase-closeouts/`：phase 收口和 rollout 过程材料
- `summaries/`：阶段性 release/summary 文档
